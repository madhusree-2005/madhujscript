 What is JavaScript?
JavaScript (JS) is a programming language mainly used to make web pages interactive.

HTML → structure (like bones)

CSS → style (like clothes)

JavaScript → behavior (like actions)

Example: When you click a button and something changes without reloading the page → that’s JavaScript.

2. How to Run JavaScript
You can run JS:

Inside an HTML file:

html

<script>
  console.log("Hello, World!");
</script>
Or in the browser console (press F12 → Console tab).

3. Variables
Variables store data.
Old ways: var (avoid)
Modern ways:

let → can be updated

const → cannot be reassigned

javascript
Copy
Edit
let name = "Alice";
const age = 20;
4. Data Types
JavaScript has basic types:

String → "Hello"

Number → 42, 3.14

Boolean → true / false

Null → nothing

Undefined → no value assigned

Object → { key: value }

Array → [1, 2, 3]

5. Operators
Math: +, -, *, /, %

Compare: ==, ===, !=, !==, <, >, <=, >=

Logical: && (and), || (or), ! (not)

6. Functions
Functions are reusable blocks of code.

javascript
Copy
Edit
function greet(name) {
  return "Hello " + name;
}

console.log(greet("Madhusree"));
